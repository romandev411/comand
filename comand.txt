 cd - перейти в папку
ls - список файлов и папок
locate, grep
sudo apt install <пакет> - установка пакета
sudo apt add ... - добавление репозитория 
sudo apt update - обновление списка пакетов из репозитория
sudo apt upgrade - обновление пакетов
CRUD - create, read, update, delete
cat <file> - просмотр файлов
less <file> - просмотр (больших) файлов
nano <file> - редактирование файла
cp <file> <destination> - копирование
mv <file> <destination> - перемещение/переименование

Открытие в фоне - <имя_процесса> &
Либо:
- открыть процесс,
- остановить его через CTRL+Z (или сразу убить через CTRL+C)

Вычислить PID через ps -x
kill <PID> - убить процесс
killall <имя_процесса> - убить все сущности этого процесса

Сигналы
-9 - безусловное убийство
-15 - обычное завершение

chmod <MODE> - смена прав доступа к файлу, где MODE:
[ugoa]([-+=])([rwx])
u - текущий юзер
g - группа юзера
o - other, другие
а - все сразу
(то есть если я говорю a=r, это значит - всем только чтение)
Цифровой эквивалент - три цифры 0-7, каждая из которых - права для определенной категории
